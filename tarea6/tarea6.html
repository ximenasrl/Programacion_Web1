Conceptos principales de HTML

HTML

Lenguaje de marcado de hipertexto, que se utiliza para hacer o estructurar una página web

¿Qúé es un elemento HTML?

Es la unidad básica de estructura y contenido de una página web, está compuesta de:

  1. Etiqueta de apertura
  2. Pares atributo-valor
  3. Contenido
  4. Etiqueta de cierre

Etiqueta <head>

Es una etiqueta que contiene metadatos, es la forma en que una página Web se comunica con el navegador, funciona para la optimización de SEO, se pueden enlazar archivos CSS o Script

Etiqueta <body>

Contiene todo el contenido html

Contenedores semánticos

<header> Etiqueta utilizada para el contenido introductorio de una página web, puede contener el logo, el menú del navegador principal, el nombre de la empresa etc.
<nav> Etiqueta utilizada para el menú de navegación principal suele contener enlaces a otras páginas relacionadas o navegar por la página principal. No debe de haber más en la página
<main> Etiqueta única de una página web cuyo propósito es resaltar nuestro objetivo o mensaje principal del sitio web. Dentro se colocará el contenido principal de nuestra página
<section> Es una etiqueta que poco a poco se intenta que sustituya a <div> su uso es para dividir la página en secciones y agrupar contenido relacionado
<article> Es una etiqueta que se utiliza para agrupar contenido autonomo que si se separa de otro contenido sigue teniendo coherencia normamente se conpone de un encabezado y contenido
<footer> Es una etiqueta que normalmente se encuentra hasta el final de una pagina funciona como cierre de un sitio web a menudo contiene datos de contacto derechos de autor u otra informacion complementaria
