Escplorando los colores

Modelos de color

RGB

Estos textos están con RBG

RGBA

Estos textos están con RBGA

Modelo HEX

Estos textos están con el modelo hexadecimal

Modelo HSL

Estos textos están con el modelo hsl(tono, saturación y brillo)

Colores multinavegador

Hay más de 160 colores con nombre, que son compatibles con todos los navegadores modernos, estos nombres se pueden usar directamente en css para aplicar colores o elementos sin especificar valores hexadecimales, rgb o hsl.

Por ejemplo:

  1. Negro
  2. Blanco
  3. Salmón
  4. Azul
  5. Verde

Colores de un borde

Para que a un elemento HTML se le pueda aplicar color a su borde son necesarias 2 propiedades: border-style y border-color

Colores degradados

Degradado lineal

El degradado lineal, es una transición de un clor a otro, de forma lineal

Degradado radial

El degradado radial, es una transición de un clor a otro, de forma radial
